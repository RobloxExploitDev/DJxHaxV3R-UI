## Installation & First Time Setup

1. Join The Discord
2. Download The UI From The UI Downloads Channel
  - Alternatively Download From the "Releases" Section of This Repo
3. Download the ZIP file 
4. Extract The ZIP
5. Make Sure your connected to internet then run the Bootstrapper. 
6. The UI Will "Check Authentication". Login with Any User & Pass. The UI Will Restart After
7. The UI Will Check for Missing Files & Downlaod Them - If Any Are Missing then the UI Will Restart After Downloading
8. Then the UI Checks If its Outdated, If it is Then it Goes through the Updating Process. Else it Continues to Main Executor Window
9. For Updating Process - UI Downlaods Updater Files Then the Updaters Ran. Then the UI Wil be Updated.

## How To Attach

1. Designed More for Single Instance Than Multi Instance
2. With Multi Instance, use /MultiInstanceHelp command In Console To Use
3. For Single Instance Open a Process And Attach.
